The transform to get the world matrix from.

direction Vector3

The direction vector to transform.

Returns

The transformed direction.

LookAt(TransformComponent, Vector3, Vector3, float)

Sets the transforms rotation so it's forward vector points at the target.

public static void LookAt(this TransformComponent transform, Vector3 target, Vector3 worldUp, float smooth = 1)

Parameters

transform TransformComponent

The TransformComponent to update.

target Vector3

The target to point towards

worldUp Vector3

A Vector specifying the upward direction.

smooth float

Value between 0 and 1 indicating the weight of target orientation. The default is 1.

Remarks

This method updates the LocalMatrix and WorldMatrix after transformation.

Exceptions

ArgumentNullException

If transform is null.

ArgumentException

If transform has UseTRS set to null.
