Remarks

This class contains utility methods designed to assist with common tasks involving vectors, such as generating random Vector3 values within a specified range.

Methods

RandomVector3(float, float, Random?)

Generates a random Vector3 with each component (X, Y, Z) within the given range. By default, min is 0 and max is required.

public static Vector3 RandomVector3(float max, float min = 0, Random? random = null)

Parameters

max float

The maximum value for X, Y, and Z components.

min float

The minimum value for X, Y, and Z components. Default is 0.

random Random

Optional random instance for custom seeding or performance.

Returns

Vector3

A random Vector3.
