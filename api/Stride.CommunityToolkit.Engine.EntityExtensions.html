AddMouseLookCamera(Entity)

Adds a mouse-look camera controller to the specified entity, enabling camera movement and rotation through various input methods.

public static void AddMouseLookCamera(this Entity entity)

Parameters

entity Entity

The entity to which the camera controller will be added.

Remarks

The camera entity can be moved using W, A, S, D, Q and E, arrow keys, a gamepad's left stick or dragging/scaling using multi-touch. Rotation is achieved using the Numpad, the mouse while holding the right mouse button, a gamepad's right stick, or dragging using single-touch.
