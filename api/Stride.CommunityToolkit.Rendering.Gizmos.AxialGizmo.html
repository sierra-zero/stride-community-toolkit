Class AxialGizmo

Namespace
Stride.CommunityToolkit.Rendering.Gizmos
Assembly
Stride.CommunityToolkit.dll

Represents a base class for creating an Axial Gizmo, which visually represents 3D axes in the scene. The Gizmo consists of three axes (X, Y, Z), each represented by a different color (Red, Green, Blue). This class provides the foundational functionality for rendering and configuring the visual aspects of the Gizmo.
